Assignment 5: Hotel Reservations
Create a Django hotel reservation application where users can view available rooms, book rooms for specific dates, and view their reservations.

Implement models for hotels, room types, and reservations.

Develop an API to view a list of hotels and available rooms, book rooms for selected dates, and view a list of their reservations.

Use Docker to containerize the application and Nginx to proxy requests to the application and serve static files.





Задание 5. Бронирование отелей
Создайте Django приложение для бронирования отелей, где пользователи могут просматривать доступные номера, бронировать номера на определенные даты и просматривать свои бронирования.

Реализуйте модели для отелей, типов номеров и бронирований.

Разработайте API для просмотра списка отелей и доступных номеров, бронирования номеров на выбранные даты и просмотра списка своих бронирований.

Используйте Docker для контейнеризации приложения, а Nginx для проксирования запросов к приложению и обслуживания статических файлов.
